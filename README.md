## ğŸ› ï¸ Installation Guide

1. Clone the repository:

    ```bash
    git clone https://github.com/erenyeager101/react-portfolio.git
    ```

2. Navigate into the project directory:

    ```bash
    cd react-portfolio
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

4. Run the development server:

    ```bash
    npm start
    ```
